# 3GPP Study Wiki

> Learn Specs → Understand Flow → Implement Software → Visualize

## Study Flow
1. 📘 Read 3GPP Spec (Original)
2. 🧠 Break down procedures
3. 💻 Design SW structures
4. 🌐 Build Web App for visualization

## Target
- 5G Core / NAS / RAN 이해
- Spec 기반 SW 설계 능력
- Streamlit / Web Dashboard 연계

## What this Wiki Delivers
- 3GPP 원문을 개발 관점에서 재구성한 요약
- 절차 중심의 단계별 해석과 상태 기계 예시
- Web App/시각화와 바로 연결할 수 있는 코드 아이디어

## Quick Start
- 좌측 내비게이션을 따라 3GPP 시리즈별로 살펴보세요.
- 각 절차는 Breakdown → SW 설계 → Web App 아이디어 순으로 이어집니다.
- GitHub Pages에서 공개 위키로 운영하기 좋게 설계했습니다.

> 이 위키는 “3GPP를 읽는 곳”이 아니라 “3GPP로 구현하는 곳”입니다.
